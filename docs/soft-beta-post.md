## Short Post
I built Sweat Streaks, a local-first macOS menu bar app for tracking daily coding streaks across GitHub, LeetCode, Codex, Claude Code, Cursor, and a combined streak.

It keeps activity data on your Mac, stores GitHub tokens in Keychain, and local AI-tool tracking uses timestamps/metadata instead of storing prompt text, chat text, edited file contents, or auth tokens.

Still early: unsigned arm64 macOS build, so macOS may require manual approval on first launch. Looking for feedback from people who care about coding streaks and local-first developer tools.

GitHub: https://github.com/anthonylu23/sweat-streaks

Install:
```bash
brew tap anthonylu23/tap
brew install --cask sweat-streaks
```
