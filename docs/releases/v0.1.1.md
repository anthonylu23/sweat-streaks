# Sweat Streaks v0.1.1

Patch release for menu-bar-only app behavior and packaging metadata.

## Highlights
- Sweat Streaks now runs as a menu-bar-only app and does not show a Dock icon while open.
- Source-built launches use accessory activation policy.
- Debug and release app bundles set `LSUIElement=true` so packaged installs launch directly as menu-bar agents.
- The completed app bundle remains ad-hoc signed during release packaging.

## Install
```bash
brew tap anthonylu23/tap
brew upgrade --cask sweat-streaks
```

Or download `Sweat-Streaks-v0.1.1-macos-arm64.zip` from GitHub Releases.

## Notes
- This release is unsigned and not notarized. macOS may require manual approval in Privacy & Security before the app opens.
- Developer ID signing, notarization, universal builds, and sanitized public screenshots remain planned follow-ups.
