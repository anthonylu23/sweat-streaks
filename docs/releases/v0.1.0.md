# Sweat Streaks v0.1.0

Initial public open-source release.

## Highlights
- macOS menu bar streak tracking for GitHub, LeetCode, Codex, Claude Code, Cursor, and Combined activity.
- Local-first SQLite persistence with GitHub PAT storage in macOS Keychain.
- Configurable provider tracking, local provider paths, menu bar streak visibility, and start-on-login support.
- Daily risk notifications and compact 13-week activity heatmaps.
- Unsigned macOS arm64 app zip for GitHub Releases and Homebrew cask distribution.

## Install
```bash
brew tap anthonylu23/tap
brew install --cask sweat-streaks
```

Or download `Sweat-Streaks-v0.1.0-macos-arm64.zip` from GitHub Releases.

## Notes
- This first release is unsigned and not notarized. macOS may require manual approval in Privacy & Security before the app opens.
- Developer ID signing, notarization, universal builds, and sanitized public screenshots are planned follow-ups.
